# odin-recipes
This is a basic recipe website I am doing to test my HTML foundations!

The recipe is for Grilled Cheese:
### Ingredients
* 2 slices of bread
* 1-2 slices of cheese
* Butter

### Steps
1. Butter the bread.
2. Place in skillet butter-side down.
3. Add cheese and second slice.
4. Grill until golden and melty
